# Contributing to NAPT

Thank you for your interest in contributing to NAPT! We welcome ideas, feedback, and bug reports.

## How to Contribute

### Feature Ideas

Have an idea for NAPT? We'd love to hear it!

1. **Check the roadmap** - See [docs/roadmap.md](roadmap.md) to see what's planned
2. **Open a GitHub Discussion** - Share your idea and get feedback
3. **Or open a GitHub Issue** - For more structured feature requests

When suggesting features, please include:

- What problem it solves
- How you envision it working
- Any examples or use cases

### Bug Reports

Found a bug? Please open a GitHub Issue with:

- NAPT version (`napt --version`)
- Python version
- Platform (Windows/Linux/macOS)
- Recipe file (or minimal example)
- Error message and traceback
- Steps to reproduce

### Questions

Have questions about using NAPT? Open a GitHub Discussion - we're happy to help!

## Code Contributions

**Note:** We're currently not accepting code contributions. If you're interested in contributing code in the future, please reach out via GitHub Discussion first.
